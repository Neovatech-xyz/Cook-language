1:test.cookにメモ帳などでコードを書く
2:ターミナルを開きcookファイルのある所へcdで移動
3:test.cookファイルの実行:python cook.py run cook file name equal test.cook
cookファイルの場所はこのファイルの親のファイル内のみで使用可能

現在確認済み使用可能OS:Windows10(Windows 10 Home Windows10バージョン22H2向け更新プログラムKB5075912 OSビルド19045.6937)
                      macOS Montereyでの動作確認予定

動作環境（開発者のパソコンのスペック）:iMac(21.5-inch, Late 2015)CPU: Intel Core i5-5575R @ 2.80GHz (4 cores)

cook 文法一覧
1. コメント
// comment

説明
→ この行は実行されない

2. 変数
create variable name equal x in 5

意味
→ x = 5

3. 定数
create constant name equal x in 5

意味
→ const x = 5

4. 出力
console.write("hello")

意味
→ 画面に出力

5. 関数
create function name equal hello arguments name equal a :
start
console.write(a)
end

呼び出し

hello("world")
6. if文
fast check if x > 10:
start
console.write("a")
end
7. if / else if / else
fast check if not x > 10:
start
console.write("a")
end

second check else if x > 5:
start
console.write("b")
end

third check else:
start
console.write("c")
end

特徴

not → 条件反転

start / end → ブロック

8. ループ
repeat this block code i > 5:
start
console.write("a")
end

意味
→ i > 5 の間繰り返す

9. インクリメント

使える書き方

i++

または

i = i + 1

または

i equal i plus 1
10. ライブラリ
use library name equal math
11. 実行コマンド
run cook file name equal main.cook
12. ファイル拡張子
.cook

mathライブラリで使える関数
use library name equal math   // 最初にこれを書く

sqrt(x) 平方根 sqrt(16) → 4.0
pi 円周率 pi → 3.14159...
floor(x) 切り捨て floor(3.7) → 3
ceil(x) 切り上げ ceil(3.2) → 4
abs(x) 絶対値 abs(-5) → 5
round(x) 四捨五入 round(3.5) → 4
sin(x) サイン sin(pi) → ≒0
cos(x) コサイン cos(0) → 1.0
tan(x) タンジェント tan(pi) → ≒0

四則演算 + % まとめ（a=10, b=3）

足し算  a plus b  a + b  13
引き算  a minus b  a - b  7
掛け算  a times b  a * b  30
割り算  a divided by b  a / b  3.333...
余り  a mod b  a % b  1
べき乗  a power b  a ** b  1000

デクリメント・インクリメント

i++    // i に 1 を足す
i--    // i から 1 を引く
もしくは英語スタイルでも書けます：

i equal i plus 1
i equal i minus 1
